# Metronome_H5
【简洁节拍器】乐器练习专用

一款纯净、简单、好用的节拍器应用

核心特点：
• 完全单机 - 无需联网，随时可用
• 零广告 - 纯净体验，专注练习
• 操作简单 - 界面清爽，一看就会
• 四种音色 - 木鱼/咔哒/电子/鼓声
• 多种拍号 - 2/4、3/4、4/4、6/8 等
• 速度范围 - 30-240 BPM 全覆盖
• 适合乐器 - 洞箫、笛子、古筝等传统乐器

打开即用，专注每一次练习。
